### javascript中常见的设计模式

设计模式就是针对某种场景下对某个问题的一种简洁而优雅的解决方案。

1. [单例模式]()

   **定义：保证一个类仅有一个实例，并提供一个访问它的全局访问点**

   场景: 登录弹窗，无论点击多少次都是弹出一个登录对话框，除外还有线程池，全局缓存、window对象
